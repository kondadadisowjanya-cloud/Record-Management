Student Record Management System
A simple, command-line interface (CLI) tool designed to manage student records efficiently using Python. This project focuses on implementing fundamental CRUD (Create, Read, Update, Delete) operations and file handling, making it an excellent exercise for beginners learning Python.

Features
Create: Add new student records, including name, age, and marks.

Read: View all stored student records.

Update: Modify existing student information.

Delete: Remove records from the system.

Persistence: Saves data automatically, ensuring your information remains available after closing the application.

Technical Concepts
This project demonstrates the application of:

Python Data Structures: Efficient use of lists and dictionaries to organize student data.

File Handling: Reading from and writing to files (CSV or text) to maintain data persistence.

CRUD Operations: Managing the lifecycle of data within an application.

Modular Programming: Organizing code into functions to improve readability and maintainability.

How to Run
Ensure Python is installed on your system.

Download/Clone the repository to your local machine.

Open your terminal or command prompt in the project directory.

Run the script using the following command:

Bash
python student_system.py
Follow the on-screen menu to navigate through the available options.

Project Structure
student_system.py: The main script containing the application logic and command-line interface.

students.csv (or .txt): The storage file where student data is saved. (This file will be created automatically upon the first save operation).
